\doxysubsection{Detailed Description}
Class containing functions for interacting with the {\bfseries{ADS7828}} Analog i2c expansion bus chip. 

This file contains the header for a class that acts as a driver for the {\bfseries{ADS7828}} 8 Channel Analog I2C expansion bus. Member functions are contained in analog\+\_\+i2c.\+cpp \begin{DoxyAuthor}{Author}

This class allows for interaction with the {\bfseries{ADS7828}} Analog I2C Expansion Bus Chip. It contains methods for joining the i2c bus as a master and reading inputs from chip. The chip does not support outputs. There is also a method to configure the chip address if a different address is used via manipulation of the address pins.
