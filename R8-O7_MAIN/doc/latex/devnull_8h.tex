\doxysubsection{Detailed Description}
This file allows debugging printouts to be conveniently turned on and off. It contains a class which doesn\textquotesingle{}t print anything when given things which are not to be printed using the {\ttfamily $<$$<$} operator, and it creates a single object called {\ttfamily Debug}. In {\bfseries{this}} {\bfseries{file}} are lines which either define or undefine the macro {\ttfamily D\+E\+B\+U\+G\+\_\+\+P\+R\+I\+N\+T\+\_\+\+O\+FF}.


\begin{DoxyItemize}
\item If the macro {\ttfamily D\+E\+B\+U\+G\+\_\+\+P\+R\+I\+N\+T\+\_\+\+O\+FF} is defined, the {\ttfamily Debug} object will be a member of class {\ttfamily \mbox{\hyperlink{classDevNull}{Dev\+Null}}} (named for the Unix file {\ttfamily /dev/null} which is sort of a black hole for data, never mind the black hole quantum information paradox) and whenever code sends information to {\ttfamily Debug} with the shift operator using code such as 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{devnull_8cpp_ab37dc5c25ebbcc889f19f2291cc4d317}{Debug}} << \textcolor{stringliteral}{"Velocity is "} << velocity << endl;}
\end{DoxyCode}
 Nothing will be printed (in fact, pretty much nothing will happen at all).
\item If the macro {\ttfamily D\+E\+B\+U\+G\+\_\+\+P\+R\+I\+N\+T\+\_\+\+O\+FF} is undefined, the {\ttfamily Debug} object will be a reference to {\ttfamily Serial} and debugging information will be printed there.
\item T\+O\+DO\+: Add the ability to send debugging information to other children of class {\ttfamily Print} such as SD cards or other serial ports.
\end{DoxyItemize}

If debugging printouts have been turned off, create an object of class {\ttfamily \mbox{\hyperlink{classDevNull}{Dev\+Null}}} which doesn\textquotesingle{}t cause anything to be printed

If debugging printouts have been turned off, create an object of class {\ttfamily \mbox{\hyperlink{classDevNull}{Dev\+Null}}} which doesn\textquotesingle{}t cause anything to be printed. This should be the only such object which ever needs to be created, unless there are several independently activated and/or deactivated debug-\/print things.
