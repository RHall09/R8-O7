\doxysubsection{Detailed Description}
Class which implements a mutex which can guard a resource. 

A mutex (short for M\+U\+Tual E\+Xclusion) is used to ensure that two tasks don\textquotesingle{}t use a resource at the same time. This is one method which can be used to prevent data corruption due to task switching. This class doesn\textquotesingle{}t add functionality to the Free\+R\+T\+OS mutex; it just simplifies the programming interface.
