Detailed Description

A class containing functions for driving a single motor using the MD-L6205D chip.

A class containing functions for driving a single motor using MD-L6205D chip. Methods include enabling, disabling, setting a desired PWM, checking for overcurrent conditions and clearing overcurrent conditions. This class does NOT implement a control system and thereby only governs motor PWM and not necessarily motor speed.

Initialize the Motor class object, create appropriate PWM channels, and initially disable motors.

Initialize the motor class object and attach it to the provided enable and PWM pins. Additionally, assign a motor number. The motor number helps differentiate different instances of Motor classes and also allows for automatic generation of PWM channels. Then, lower the enable pin to disable the motor until the enable method is called.

Parameters
Pin_EnThe pin number of Pin object to which the motor enable pin is attached.
Pin_1The pin number or Pin object to which the first motor PWM pin is attached.
Pin_2The pin number or Pin object to which the second motor PWM pin is attached.
mot_numA unique number assigned to the motor for generating unique PWM channels and motor identification.
